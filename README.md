# Employee Attiretion Project
The project focuses on analyzing company-level data and building a professional business intelligence dashboard using Power BI. Data transformation and preprocessing were performed using Power Query, while DAX was used to develop calculated measures and KPIs.  The dashboard includes multiple visualizations such as revenue distribution, company rankings, sector analysis, and performance comparisons. These visuals help uncover patterns in company growth, profitability, and workforce productivity.  The objective of the project is to demonstrate how data analytics and visualization tools can convert complex datasets into clear and actionable insights for business stakeholders.
