# Employee Attiretion Project
## 📊 Project Overview

This project presents an interactive Power BI dashboard designed to analyze company performance using key financial and operational metrics. The dashboard enables users to explore revenue trends, profitability, market capitalization, and workforce distribution across companies.

By transforming raw data into meaningful visual insights, the project helps identify top-performing organizations, sector-wise trends, and efficiency indicators. The dashboard provides a comprehensive company-level view that supports data-driven decision-making.

---

## 📌 Project Description

The project focuses on analyzing company-level data and building a professional business intelligence dashboard using Power BI. Data transformation and preprocessing were performed using Power Query, while DAX was used to develop calculated measures and KPIs.

The dashboard includes multiple visualizations such as revenue distribution, company rankings, sector analysis, and performance comparisons. These visuals help uncover patterns in company growth, profitability, and workforce productivity.

The objective of the project is to demonstrate how data analytics and visualization tools can convert complex datasets into clear and actionable insights for business stakeholders.

---

## 🎯 Project Goals

- Analyze company performance using financial and operational data  
- Identify key trends in revenue, profitability, and workforce efficiency  
- Build an interactive dashboard for data exploration and analysis  
- Enable stakeholders to make data-driven business decisions  

---

## 📊 Key Insights

- Companies with higher employee counts generally generate higher revenue, indicating strong scalability.
- Revenue per employee is significantly higher in top-ranked companies.
- Market capitalization strongly correlates with profitable organizations.
- Sector-wise analysis highlights dominant industries contributing to total revenue.
- Companies experiencing rank improvements often demonstrate consistent revenue growth.
